Installation

Install the Aidni CLI to manage your clusters directly from the terminal.

npm install -g @aidni/cli
# or
pip install aidni-cli

Authentication

Authenticate your session using your API key found in the dashboard settings.

aidni login --key YOUR_API_KEY

Clusters API

Manage your compute clusters and monitor their health status.

GET /v1/clusters/{cluster_id}/health

Returns the real-time health metrics of a specific cluster.

{
  "status": "operational",
  "uptime": "99.99%",
  "gpu_utilization": 85,
  "active_nodes": 12
}

Python SDK

Interact with Aidni programmatically using our Python client.

from aidni import AidniClient

client = AidniClient(api_key="...")

# Get all active models
models = client.models.list(status="training")

for model in models:
    print(f"Model: {model.name}, Loss: {model.current_loss}")
